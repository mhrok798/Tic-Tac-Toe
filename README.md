# Tic-Tac-Toe

## 仕様

- 動作プラットフォーム
  - PC等
- 開発プラットフォーム
  - Visual Studio Code
- 開発言語
  - Java
- 利用ライブラリ
  - Java swing
  - Java awt

## 使用方法
 JDKをインストールし、javacコマンドを使用してソースコードをバイトコード(.classファイル)にコンパイルする。javacコマンドを使って生成された.classファイルを実行すると三目並べが開始される。プレイヤーは交互にマスを選択し、縦・横・斜めいずれかが揃うとゲームが終了し、勝敗が表示される。揃う前に全てのマスが埋まると引き分けと表示される。
